###Goal:

  To quickly compile and/or run any file you want with one command. 

###Installation:

```sh
$ git clone https://github.com/mjtp/electric.git
$ cd electric
$ npm install -g
$ npm link
```
  
###Usage:

```sh
$ electric < filepath >
```
